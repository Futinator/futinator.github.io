Frequently Asked Questions (FAQ)

  • What is Futinator?

    • Futinator is a powerful chrome extension that helps to handle the FIFA Ultimate Team (FUT) web app easier. You can bind actions like quick buy, search, increase/decrease price and many more to keys on your keyboard. This allows you to snipe as quickly as possible

  • Can I get banned using Futinator?

    • Futinator exists since Fifa 19 and there has never been a ban!

  • How can I get Futinator?

    • You can download Futinator in the Google Chrome Web Store.

  • Futinator no longer works, what should I do?

    • Normally that will never happen. But should it not work for you, you need not panic. Try the following steps:
